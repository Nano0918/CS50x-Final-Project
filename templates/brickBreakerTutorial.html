Brick Breaker Tutorial

  • Press Start to begin the game
  • Press either (W,D) or Right and Left Arrow keys to move the paddle
  • In order to win break all the bricks at the top of the screen
  • Once you've won try and see if you can beat it again
  • The layout changes slightly so it's a new challenge everytime
